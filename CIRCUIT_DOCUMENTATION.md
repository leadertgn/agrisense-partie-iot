# Documentation du Projet Agrisense IoT

## 1. Vue d'ensemble

Ce projet vise à automatiser la surveillance et le contrôle d'un système d'irrigation. Il utilise un microcontrôleur ESP8266 pour lire les données de capteurs d'humidité de l'air, de température et d'humidité du sol. Ces données sont ensuite envoyées à une base de données Firebase en temps réel. Le système peut également contrôler une électrovanne (via un relais) en fonction des données reçues de Firebase, permettant une irrigation automatisée ou contrôlée à distance. L'architecture repose sur la lecture de capteurs locaux, la communication WiFi, la synchronisation horaire NTP, et l'interaction avec Firebase pour la télémétrie et le contrôle.
